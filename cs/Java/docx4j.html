# docx4j

用于读写控制docx文件

# API

加载文档

WordprocessingMLPackage wordMLPackage = WordprocessingMLPackage.load(new java.io.File(path));

获取MainDocumentPart对象

MainDocumentPart documentPart = wordMLPackage.getMainDocumentPart();

可通过 documentPart.getXML()获取文档XML格式的内容，例如:

<w:document mc:Ignorable="w14 w15 wp14" 
xmlns:dsp="http://schemas.microsoft.com/office/drawing/2008/diagram" 
...
    <w:body>
        <w:p>
            <w:pPr>
                <w:ind w:firstLine="500" w:firstLineChars="0"/>
                <w:rPr>
                    <w:highlight w:val="none"/>
                </w:rPr>
            </w:pPr>
            <w:r>
                <w:rPr>
                    <w:color w:val="FF0000"/>
                    <w:highlight w:val="none"/>
                </w:rPr>
                <w:t>峰乌尔回复，谁不服较</w:t>
            </w:r>
            <w:bookmarkEnd w:id="0"/>
            <w:r>
                <w:rPr>
                    <w:highlight w:val="none"/>
                </w:rPr>
                <w:t>为方便玩家发表</w:t>
            </w:r>
        </w:p>
        <w:sectPr>
            <w:pgSz w:w="11907" w:h="16839"/>
            <w:pgMar w:top="1440" w:right="1440" w:bottom="1440" w:left="1440" w:header="720" w:footer="720" w:gutter="0"/>
        </w:sectPr>
    </w:body>
</w:document>

可通过documentPart.getContents()方法获取 Document对象，该对象的getContent()方法可获取段落等内容的Object对象的集合。

Document wmlDocumentEl = documentPart.getContents();
List list = wmlDocumentEl.getContent();

可以通过 o instanceof org.docx4j.wml.P 这种方式判断后，强制转换成目标对象。例如：

if (o instanceof org.docx4j.wml.P) {
    P p = ((P) o);
}

P对象段落对象对应XML中 <w:p> 标签中的内容。可以观察到P对象中含有pPr对象和R对象。同样

// 获取R对象 
List list = ((P) o).getContent();
for (Object o1 : ((P) o).getContent()) { 
    if (o1 instanceof org.docx4j.wml.R) {
        R r = (R) o1;
    }
}
// 获取PPr对象,PPr对象包含了该段落的高亮色等
((P) o).getPPr();

通过观察我们同样发现R对象中有RPr对象和T对象

// 获取RPr对象
RPr rpr = r.getRPr();

// 获取Color对象
Color Color = rpr.getColor();

// 获取T对象
for (Object o2: r.getContent()){
    Text t = (Text) ((JAXBElement)o2).getValue();
}
