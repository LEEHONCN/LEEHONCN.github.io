    # 块级元素与行内元素

    块级元素

    • 独占一行，默认情况下自动填满父元素宽度，即使设置宽度也会独占一行
    • 可以设置 width height
    • 有margin padding 属性
    • display：block

    行内元素

    • 不会独占一行，相邻行内元素会排列在同一行里，排不下才会换行，其宽度随元素的内容而变化
    • width height 无效
    • margin padding 水平方向有效 ，垂直方向无效
    • display：inline

    vertical-align设置元素的垂直对齐方式。

    # 尺寸

    属性 描述
    height 设置元素的高度。
    line-height 设置行高。
    max-height 设置元素的最大高度。
    max-width 设置元素的最大宽度。
    min-height 设置元素的最小高度。
    min-width 设置元素的最小宽度。
    width 设置元素的宽度。

    # 分类属性

    属性 描述
    clear 设置一个元素的侧面是否允许其他的浮动元素。
    cursor 规定当指向某元素之上时显示的指针类型。
    display 设置是否及如何显示元素。
    float 定义元素在哪个方向浮动。
    position 把元素放置到一个静态的、相对的、绝对的、或固定的位置中。
    visibility 设置元素是否可见或不可见。

    # CSS长度

    绝对长度

    • in 英寸

    • cm 厘米

    • mm 毫米

    • pt 1/72英寸

    • pc 12 pt

    相对长度

    • % 相对于上一级元素的大小

    • px 的问题在于对于不同的手机屏幕，手机的像素密度不同。因此呈现出来五花八门。

    • 字体相对长度

      • em(相对于当前对象内文本的字体尺寸) 当出现层级情况，计算复杂。而且其中一个层级变化，会引起所有上层的元素重绘。
      • rem(相对于根元素确定)
      • ex 依赖于英文小写字母x的高度
      • ch 数字0的宽度

    • 视窗相对长度

      • vw 1vw = 视窗宽度的1%
      • vh 视窗高度
      • vmin vw和vh中较小的那个
      • vmax vw和vh中较大的那个
