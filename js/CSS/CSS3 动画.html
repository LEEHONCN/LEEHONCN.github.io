    # CSS3 动画

    transition：过渡

    ​ 需要事件触发，一次性，只有开始和结束两种状态

    • transition-property :

      • none 没有属性
      • all 所有属性
      • property 定义属性列表

    • transition-duration

      • time 默认为0 秒(s)或毫秒

    • transition-timing-function

      • linear 规定以相同速度开始至结束的过渡效果（等于 cubic-bezier(0,0,1,1)）。
      • ease 规定慢速开始，然后变快，然后慢速结束的过渡效果（cubic-bezier(0.25,0.1,0.25,1)）。
      • ease-in 规定以慢速开始的过渡效果（等于 cubic-bezier(0.42,0,1,1)）。
      • ease-out 规定以慢速结束的过渡效果（等于 cubic-bezier(0,0,0.58,1)）。
      • ease-in-out 规定以慢速开始和结束的过渡效果（等于 cubic-bezier(0.42,0,0.58,1)）。
      • cubic-bezier(*n*,*n*,*n*,*n*) 在 cubic-bezier 函数中定义自己的值。可能的值是 0 至 1 之间的数值。

    • transition-delay

      • time
    transition: property duration timing-function delay;

    animation：动画

    transition的升级版，可以定义多种属性

    • @keyframes 定义动画规则

    • animation-name

      • @keyframesname
      • none 用于覆盖从级联的动画

    • animation-duration 动画持续时间

      • time

    • animation-timing-function ： 同 transition-timeing-function

    • animation-delay

      • time

    • animation-iteration-count：规定动画播放几次

      • n 播放次数
      • infinite 无限次

    • animation-direction 播放方向

      • normal 正常播放
      • reverse 反向播放
      • altermate 奇数次正常播放，偶数次反向播放
      • altermate-reverse 偶数次正常播放，奇数次反向播放

    • animation-fill-mode

      • none 不应用
      • forwards 应用最后一次动画的属性值
      • backwards 应用第一次定义的动画属性值
      • both 同时遵循 forwards 和 backwards

    • animation-play-state 指定动画是否正在运行或已暂停

      • paused
      • running
    animation: name duration timing-function delay iteration-count direction play-state fill-mode;

    transform： 向元素应用2D或3D转换

    • 2D转换

      • translate(x,y) 根据左和顶部移动

        • translateX
        • translateY

      • rotate(ndeg) 顺时针旋转

      • scale(x,y) 水平和垂直扩大的倍数

        • scaleX
        • scaleY

      • skew(deg,deg) 在水平，垂直方向扩大的倍数

        • skewX
        • skewY

      • matrix(a,b,c,d,e,f) 矩阵变换
