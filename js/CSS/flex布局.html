    # flex布局

    有容器和轴两个概念

    首先需要父容器要设置display：flex

    • 父容器属性：

      • flex-direction ：属性确定主轴的方向，交叉轴由主轴方向确定。

        交叉轴由主轴逆时针方向旋转90°得到。

        轴的起始端和末尾端分别用flex-start和flex-end表示

        • row：向右
        • column：向下
        • row-reverse：向左
        • column-reverse：向上

      • justify-content : 沿着主轴方向排列子容器

        • flex-start ：靠左
        • flex-end：靠右
        • center：水平居中
        • space-between：左右贴边，中间距离平均
        • space-around：左右留边，中间距离是侧边距离的2倍

      • align-content：多行沿着交叉轴方向排列子容器

        • flex-start ：靠左
        • flex-end：靠右
        • center：水平居中
        • space-between：左右贴边，中间距离平均
        • space-around：左右留边，中间距离是侧边距离的2倍
        • stretch：拉伸对齐

      • align-items : 沿着交叉轴方向分配子容器（垂直排列）

        • flex-start：靠上
        • flex-end：靠下
        • center：垂直居中
        • baseline：首行文字对齐
        • stretch：沿垂直方向拉伸高度至于父元素一致

      • flex-wrap：决定子元素是否换行排列

        • nowrap：不换行 默认
        • wrap：换行
        • wrap-reverse：反向换行

      • flex-flow：flex-direction和flex-wrap的复合属性，也可以只设置一个，也可以同时设置。

    • 子容器属性：

      • align-self：在交叉轴上如何排列
        • 同 align-items一致，align-self优先级较高
      • flex-grow：子容器弹性伸展比例，剩余空间按比例分配给子容器。
        • 数字
      • flex-shrink：子容器弹性收缩比例
        • 数字
      • flex-basis： 同width，优先级比width高。若都为auto则非auto的属性优先级更高
        • 长度单位
      • flex：在主轴上如何伸缩，其实就是flex-grow、flex-shrink、flex-basis组合简写
        • 默认值 0 1 auto
        • 单值
          • 有单位 basis
          • 无单位 grow
        • 双值
          • 有单位 grow basis
          • 无单位 grow shrink
        • 三值 grow shrink basis
      • order：改变子容器的排列顺序，覆盖html代码中的顺序，默认值为0，可以为负值，数值越小越靠前

    总结：

    flex布局主要有两个概念：容器和轴。其中轴是在父容器中确定的。利用flex布局可以很好地控制子元素在父容器中的位置排列。

    父容器里可以设置

    • 主轴和交叉轴的方向

    • 子容器在父容器中的主轴排列方式，

    • 子容器在父容器中的交叉轴排列方式

    • 多行子容器在父容器中的交叉轴排列方式

    • 多行子元素是否换行

    子容器里可以设置

    • 在交叉轴上如何排列
    • 排列顺序
    • 子容器的宽度
    • 子容器伸长的比例
    • 子容器收缩的比例
