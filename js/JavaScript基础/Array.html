          # Array

          # Array 属性

          Array.length： 总是大于数组最大项得下标

          Array.prototype： 可以用它向原型添加方法

          # Array 构造器

          Array

          let a = Array();
let b = Array(8);
let c = Array(['1', '2', '3']);
let d = Array('1', '2', '3');

// []
// [ <8 empty items> ]
// [ [ '1', '2', '3' ] ]
// [ '1', '2', '3' ]

          Array.of 与 Array不同的是

          Array.of(7)    // [7]

          Array.from 从一个类似数组的或可迭代对象中创建一个新的数组

          Array.from(arrayLike[, mapFn[, thisArg]])

          # Array 判断

          Array.isArray var a = [];

          • 基于instanceof a instanceof Array;

          • 基于constructor a.constructor === Array;

          • 基于Object.prototype.isPrototypeOf Array.prototype.isPrototypeOf(a);

          • 基于getPrototypeOf Object.getPrototypeOf(a) === Array.prototype;

          • 基于Object.prototype.toString Object.prototype.toString.apply(a) === '[object Array]';

          # 改变自身的方法

          # 栈方法

          pop 栈顶弹出

          arr.pop()

          返回从数组中删除的元素(当数组为空时返回undefined)。

          push 栈顶推入

          arr.push(element1, ..., elementN)

          返回数组长度

          # 队列方法

          shift shift()方法删除数组的第一个元素，并返回这个元素。 栈底弹出。返回从数组中删除的元素(当数组为空时返回undefined)。

          arr.shift()

          unshift unshift() 方法用于在数组开始处插入一些元素，并返回数组新的长度。 栈底插入。 返回数组长度。

          arr.unshift(element1, ..., elementN)

          # 排序和倒序

          reverse 数组倒置，返回数组的引用。 无参数

          arr.reverse()

          sort 排序默认是根据Unicode来排序的，所以排序数字的时候会出现 10 80 9 这样的情况，需要指定比较函数。

          arr.sort([compareFunction])

          # 改变内容

          splice 从某个位置开始删除n个元素，并插入m个元素。 返回由被删除的元素组成的一个数组。

          array.splice(start[, deleteCount[, item1[, item2[, ...]]]])

          copyWithin 方法浅复制数组的一部分到同一数组中的另一个位置，并返回它。 返回改变后的数组。

          arr.copyWithin(target[, start[, end]])

          fill 用固定的值覆盖 起始到终止位置。 返回改变后的数组。

          arr.fill(value[, start[, end]])

          # 不会改变自身的方法

          # 合并与拆分

          concat： 方法用于合并两个或多个数组。此方法不会更改现有数组，而是返回一个新数组。

          var new_array = old_array.concat(value1[, value2[, ...[, valueN]]])

          slice 切分数组。slice() 方法将数组中一部分元素浅复制存入新的数组对象，并且返回这个数组对象。 slice方法参数为空时，同concat方法一样，都是浅复制生成一个新数组。

          slice([begin [, end]])

          # 返回字符串

          join 将数组按指定字符串拼接

          let str = arr.join([separator])

          toString Array对象覆盖了Object的 toString 方法。对于数组对象，toString 方法连接数组并返回一个字符串，其中包含用逗号分隔的每个数组元素。

          toLocalString 返回一个字符串表示数组中的元素。数组中的元素将使用各自的 toLocaleString 方法转成字符串，这些字符串将使用一个特定语言环境的字符串（例如一个逗号 ","）隔开。

          toSourse 返回一个字符串,代表该数组的源代码.

          # 查找元素

          # 二叉树查找

          **indexOf ** 首个被找到的元素在数组中的索引位置; 若没有找到则返回 -1

          arr.indexOf(searchElement[, fromIndex = 0])

          lastIndexOf 逆向查找

          arr.lastIndexOf(searchElement[, fromIndex = arr.length - 1])

          includes 返回值为boolean

          arr.includes(valueToFind[, fromIndex])
          # 遍历查找

          find 返回第一满足的元素的值，没有则undefined

          arr.find(callback(element[, index[, array]])[, thisArg])

          findIndex 返回第一满足的元素的索引，没有则 -1

          arr.findIndex(callback(element[, index[, array]])[, thisArg])

          # 扁平化

          flat 将返回的数组扁平化

          var newArray = arr.flat(depth)

          flatMap 将返回的数组扁平化并将每一项按函数处理

          var new_array = arr.flatMap(function callback(currentValue[, index[, array]])[, thisArg])

          # 全部或存在满足

          every 每个元素通过测试则为true

          arr.every(callback(item[, index[, array]])[, thisArg]);

          some 与every相对

          arr.some(callback(item[, index[, array]])[, thisArg]);

          # 筛选，同一操作，累算

          filter 返回通过测试的元素数组

          var newArray = arr.filter(callback(element[, index[, array]])[, thisArg])

          map 每个元素做一次同一操作

          var newArray = arr.map(callback(element[, index[, array]])[, thisArg])

          reduce 累算

          arr.reduce(callback(accumulator, currentValue[, index[, array]])[, initialValue])

          reduceRight 从右往左累算

          # 迭代器

          entries 返回一个新的Array Iterator对象，该对象包含数组中每个索引的键/值对。

          arr.entries()

          keys **keys()**方法返回一个包含数组中每个索引键的**Array Iterator**对象。

          values values() 方法返回一个新的 Array Iterator 对象，该对象包含数组每个索引的值

          Symbol.iterator 数组的 iterator 方法，默认情况下与 values() 返回值相同

          arr[Symbol.iterator]()

          forEach 不能跳出循环

          arr.forEach(callback(item[, index[, array]])[, thisArg]);
