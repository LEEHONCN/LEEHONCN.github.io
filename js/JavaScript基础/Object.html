          Object.prototype

          Object.prototype.proto 获取对象内部 [[Prototype]]

          Object.prototype.constructor 返回创建实例对象的Object构造函数的引用

          # 方法

          # 定义对象方法

          # 原型相关

          Object.create 类式继承 第二个属性可以是 Object.getOwnPropertyDescriptors 的返回值。

          Object.create(proto, [propertiesObject])

          Object.getPrototypeOf 方法返回指定对象的原型（内部[[Prototype]]属性的值）

          Object.getPrototypeOf(obj)

          Object.setPrototypeOf 方法设置一个指定的对象的原型 ( 即, 内部[[Prototype]]属性）到另一个对象或 null。

          Object.setPrototypeOf(obj, prototype)

          # 属性描述对象相关

          Object.defineProperty 定义属性

          Object.defineProperty(obj, prop, descriptor)

          Object.defineProperties 定义多个属性

          Object.defineProperties(obj, propertiesObject)

          Object.getOwnPropertyDescriptor 方法返回指定对象上一个自有属性对应的属性描述符。

          Object.getOwnPropertyDescriptor(obj, prop)

          # 枚举属性方法

          Object.keys

          方法会返回一个由一个给定对象的自身可枚举属性名组成的数组，数组中属性名的排列顺序和使用 for...in 循环遍历该对象时返回的顺序一致 。如果对象的键-值都不可枚举，那么将返回由键组成的数组。

          Object.keys(obj)

          Object.values

          方法返回一个给定对象自身的所有可枚举属性值的数组，值的顺序与使用for...in循环的顺序相同 ( 区别在于 for-in 循环枚举原型链中的属性 )。

          Object.values(obj)

          Object.entries 返回一个给定对象自身可枚举属性的键值对数组，其排列与使用 for...in 循环遍历该对象时返回的顺序一致

          Object.fromEntries 将键值对转换为对象 Object.entries逆操作

          Object.getOwnPropertyNames 返回一个由指定对象的所有自身属性的属性名（包括不可枚举属性但不包括Symbol值作为名称的属性）组成的数组。与 Object.keys 相似，但是它返回不可枚举属性

          Object.getOwnPropertyNames(obj)

          Object.getOwnPropertySymbols 返回一个给定对象自身的所有 Symbol 属性的数组。

          Object.getOwnPropertySymbols(obj)

          # 原型方法

          # 判断是否是自身，是否是原型，是否可枚举

          Object.prototype.hasOwnProperty

          方法会返回一个布尔值，指示对象自身属性中是否具有指定的属性（也就是是否有指定的键）

          obj.hasOwnProperty(prop)

          唯一一个不会遍历原型链的方法

          **Object.prototype.isPrototypeOf **

          判断该对象是否为参数对象的原型。

          prototypeObj.isPrototypeOf(object)

          Object.prototype.propertyIsEnumerable

          方法返回一个布尔值，表示指定的属性是否可枚举。

          obj.propertyIsEnumerable(prop)

          # 类型转换

          Object.prototype.toLocalString 方法返回一个该对象的字符串表示

          obj.toLocaleString();

          Object.prototype.toSource 方法返回一个表示对象源代码的字符串。

          Object.toSource();

          Object.prototype.toString 方法返回一个表示该对象的字符串。

          object.toString()

          Object.prototype.valueOf 方法返回指定对象的原始值。

          object.valueOf()

          # 限制对象

          Object.seal 封闭一个对象，阻止添加新属性并将所有现有属性标记为不可配置。当前属性的值只要可写就可以改变。

          Object.seal(obj)

          Object.isSealed 方法判断一个对象是否被密封。

          Object.isSealed(obj)

          Object.preventExensions 方法让一个对象变的不可扩展，也就是永远不能再添加新的属性。

          Object.preventExtensions(obj)

          Object.isExtensible 判断一个对象是否是可扩展的（是否可以在它上面添加新的属性）。

          Object.isExtensible(obj)

          Object.freeze 冻结对象，不能修改

          Object.freeze(obj)

          Object.isFrozen 方法判断一个对象是否被冻结。

          Object.isFrozen(obj)

          # ES6新增的工具

          Object.is 判断两个值是否是相同的值

          Object.is(value1, value2);

          在之前Js只有两个运算符比较两个值是否相等，但是 严格相等不能很好的处理两种特殊情况即NaN和 +0 ，-0的情况。Object,is 可以很好的处理这种情况。于是我们比较相等就有了三种方法。 ==, ===, Object.is

          Object.assign

          Object.assign(target, sourceObj1, sourceObj2, ...);

          • 将所有自身的，可枚举属性从一个或多个源对象复制到目标对象，并返回目标对象。

          • 属性的拷贝是浅拷贝，源对象属性对象改变将反应到目标对象上。

          • 同名属性会被向后替换。

          • 数组会被视为对象。属性名为 0，1，2…

          • 取值函数将会被求值之后再复制。

          # 应用场景
          • 为对象添加属性
          Object.assign(this, a, b);
// 而不是
this.a = a;
this.b = b;
          • 为对象添加方法
          Object.assign(SomeClass.prototype, {
  Method1(){},
  Method2(){}
});
// 而不是
SomeClass.prototype.Method1(){};
SomeClass.prototype.Method2(){};
          • 克隆对象
          Object.assign({}, origin);
          • 指定默认值
          Object.assign({}, Default, options);

          Object.getOwnPropertyDescriptors

          方法返回指定对象上所有自身属性对应的属性描述符。

          Object.getOwnPropertyDescriptors(obj)

          什么？ Object.assign 不能拷贝 get set ?

          const obj = {
  foo: 123,
  get bar() { return 'abc' }
};
const target = {}；
Object.assign(target, obj);
Object.defineProperties(target, Object.getOwnPropertyDescriptors(obj));

          搭配Object.create() 创建新对象
